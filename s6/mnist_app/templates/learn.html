What is MNIST?

The MNIST dataset is a collection of 70,000 handwritten digits (0-9) that has become a standard benchmark for testing machine learning algorithms. Each image is 28x28 pixels in grayscale, making it perfect for learning and testing digit recognition systems.

Our Model Architecture

We've implemented a custom Convolutional Neural Network (CNN) architecture that achieves high accuracy while maintaining computational efficiency. Here's a breakdown of our model:

Convolutional Layers

Multiple convolutional layers with ReLU activation for feature extraction

Pooling Layers

Max pooling layers to reduce spatial dimensions and extract important features

Fully Connected Layers

Dense layers with dropout for final classification

class MNISTNet(nn.Module):
    def __init__(self):
        super(MNISTNet, self).__init__()
        self.conv1 = nn.Conv2d(1, 32, 3, 1)
        self.conv2 = nn.Conv2d(32, 64, 3, 1)
        self.dropout1 = nn.Dropout2d(0.25)
        self.dropout2 = nn.Dropout2d(0.5)
        self.fc1 = nn.Linear(9216, 128)
        self.fc2 = nn.Linear(128, 10)
